jSignature
==========

Add to your gemfile 
```
gem "jSignature", "~> 0.0.2"
```

and run

```
bundle install
```

or install manually

```
gem install jSignature
```




You can use helper method to add sign field to form:

```
<%= sign_form_field(:name [, height [, width] ] ) %>
```
